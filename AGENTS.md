# Project Instructions

This repository is a quantitative market analysis framework.

Main focus:
- SMC / ICT
- RTM
- Liquidity
- Regime Detection
- Multi-timeframe analysis

Rules:
- Avoid simplistic indicators
- Avoid repainting logic
- Avoid future data leakage
- Keep code modular and clean
- Validate all dataframe operations

Structure:
/core
/data
/features
/liquidity
/regime
/tests

Coding rules:
- Use vectorized pandas operations
- Avoid large monolithic scripts
- Always include basic validation checks
- Prefer small reusable functions
